# 《TA 适合结婚吗？》关系评估问卷

## 开场文案

婚姻不是“够不够爱”这么简单。
它更像一场长期合作：要看安全感、边界感、责任感，也要看你们能不能一起处理现实。

这份问卷会从五个维度评估你们的关系状态：

* 安全与底线
* 互动与情绪模式
* 价值观与现实共识
* 冲突与修复能力
* 家庭边界与长期风险

**请根据你最近 3-6 个月的真实感受作答。**
没有标准答案，选“最接近”的即可。
如果你对某些情况并不清楚，请放心选择“我不确定 / 还没看清”。
